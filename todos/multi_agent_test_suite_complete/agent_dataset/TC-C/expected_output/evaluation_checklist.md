## 失败模式识别

| 失败现象 | 说明的问题 |
|---------|-----------|
| 只有一篇文章（正方或反方） | 并发子任务中有一个 Agent 的输出丢失，Merge Agent 没有等待所有子任务完成 |
| 反方在前，正方在后 | Merge Agent 合并时顺序错误，没有遵循指令 |
| 两篇文章都存在，但没有合并为一个文件（分别是两个文件） | Merge Agent 没有被触发，或者 Orchestrator 没有规划合并步骤 |
| 文章内容相互矛盾但论据相同 | 两个子 Agent 共享了上下文，互相影响（隔离性问题） |
| 总标题缺失 | Merge Agent 只做了拼接，没有按格式要求添加标题 |
