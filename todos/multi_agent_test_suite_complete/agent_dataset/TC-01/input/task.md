# 任务指令 | TC-01：多 Agent 协作信息传递测试

## 测试维度
多 Agent 协调（Inter-Agent Coordination）

## 你的任务
帮我策划一个"智能咖啡机"的众筹项目。请按照以下步骤完成，**每一步的结果必须作为下一步的输入**：

**步骤 1 — 市场调研**
搜索目前市面上销量前三的智能咖啡机产品，整理它们的核心卖点（价格区间、主打功能、用户评价关键词）。

**步骤 2 — 众筹文案撰写**
根据步骤 1 的调研结果，撰写一篇 500 字左右的众筹文案。文案必须：
- 突出我们产品的差异化优势：**支持语音控制（Voice Control）**
- 明确指出竞品的不足之处
- 包含一个吸引人的产品口号（Slogan）

**步骤 3 — 网页落地页开发**
根据步骤 2 生成的文案，创建一个 HTML 落地页文件，保存为 `landing_page.html`。要求：
- 网页中的所有正文内容必须使用步骤 2 生成的真实文案，**不得使用 Lorem ipsum 或任何占位文字**
- 页面结构包含：标题区、产品介绍区（含文案）、一个 `<div id="product-image">` 占位区（用于放产品图）、底部众筹按钮
- 使用内联 CSS，无需外部依赖

## 最终交付物
请在完成后提供以下三个文件：
1. `research_report.md` — 竞品调研报告
2. `crowdfunding_copy.md` — 众筹文案
3. `landing_page.html` — 网页落地页

## 评分关键点（供参考）
- landing_page.html 中的文案内容必须与 crowdfunding_copy.md 一致
- 三个文件必须全部存在
- HTML 文件可以在浏览器中正常打开，无语法错误
