## 失败模式识别

| 失败现象 | 说明的问题 |
|---------|-----------|
| 输出了 Markdown 表格而非 HTML 文件 | Agent 遗忘了约束 3（最常见失败） |
| HTML 文件存在但某天没有动漫景点 | Agent 在长链条执行中遗忘了约束 1 |
| 有景点但没有午餐安排 | Agent 遗忘了约束 2 |
| 只在对话框中输出了 HTML 代码，没有保存文件 | Agent 不会使用文件写入工具 |
