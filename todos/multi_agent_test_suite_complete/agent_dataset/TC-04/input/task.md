# 任务指令 | TC-04：长链条任务与隐性约束记忆测试

## 测试维度
任务完成率（Task Success Rate）

## 你的任务
请为我规划一个为期 3 天的东京旅行行程，并生成一个 HTML 文件。

## 强制约束（全部必须满足）

**约束 1**：每天的行程中，必须包含**至少一个动漫或二次元相关的景点**（例如：秋叶原、吉卜力美术馆、池袋乙女路、漫画博物馆等）。

**约束 2**：每天必须安排一个**午餐餐厅**，该餐厅必须位于当天动漫景点**步行 15 分钟以内**的范围，并注明餐厅名称和大致地址。

**约束 3**：最终输出**必须是一个 HTML 文件**，文件名为 `tokyo_trip.html`。内容使用 HTML 表格呈现，**不得输出 Markdown 格式**，不得只在对话框中回复代码。

## 行程格式要求
`tokyo_trip.html` 中的表格应包含以下列：

| 天数 | 上午 | 午餐（餐厅名 + 地址） | 下午 | 晚上 |
|------|------|----------------------|------|------|

## 最终交付物
1. `tokyo_trip.html` — 可在浏览器中打开的旅行行程表

## 评分关键点（供参考）
- 3 天行程全部完整
- 每天都有动漫景点（约束 1）
- 每天都有午餐安排且在景点附近（约束 2）
- 输出为 .html 文件（约束 3）
