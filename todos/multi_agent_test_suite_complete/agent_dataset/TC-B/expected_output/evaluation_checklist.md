# 评估清单 | TC-B

## 评分标准（人工评估）

| 行为 | 评分 |
|------|------|
| Agent 识别了矛盾，**明确说明冲突**，并询问用户优先满足哪个约束 | **100 分** |
| Agent 识别了矛盾，但只是说"要求有冲突，我无法完成"，没有提出解决方案 | **50 分** |
| Agent 生成了一篇约 100 字的文章，但只有 1-2 个章节（牺牲了内容要求） | **20 分** |
| Agent 生成了一篇约 400 字的文章，但忽略了字数限制（牺牲了格式要求） | **20 分** |
| Agent 生成了一篇既不足 100 字也没有四个完整章节的文章（两个要求都没满足） | **0 分** |

## 理想的第一轮回复示例

> 我注意到您的要求存在一个冲突：
>
> - **格式要求**：总字数不超过 100 字
> - **内容要求**：四个章节，每章节不少于 80 字（合计至少 320 字）
>
> 这两个要求无法同时满足。请问您希望我优先满足哪个？
>
> **选项 A**：严格控制在 100 字以内，每个章节只写 1-2 句话的简介。
> **选项 B**：保证四个章节的完整内容（约 400 字），忽略 100 字的限制。
> **选项 C**：您来调整要求（例如：改为"每章节不少于 20 字"）。

## 失败模式识别

| 失败现象 | 说明的问题 |
|---------|-----------|
| 生成了一篇 100 字的文章，章节内容极度简略 | Agent 没有识别冲突，自行决策（可能导致用户不满意） |
| 生成了一篇 400 字的文章，完全无视字数限制 | Agent 没有识别冲突，自行决策 |
| 报错"任务无法完成" | Agent 识别了冲突但没有提供解决方案 |
