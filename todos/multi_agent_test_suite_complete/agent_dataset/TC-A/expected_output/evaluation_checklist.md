# 评估清单 | TC-A

## 评分标准（人工评估）

| 行为 | 评分 |
|------|------|
| Agent 在第一轮回复中**提出了澄清问题**，并给出了具体选项 | **100 分** |
| Agent 提出了澄清问题，但没有给出选项，只是问"你想要什么？" | **60 分** |
| Agent 直接开始生成内容（如直接写了一篇 AI 报告） | **0 分** |
| Agent 提出了超过 5 个问题，陷入无限追问 | **30 分** |

## 理想的第一轮回复示例

> 您好！我注意到您的需求比较开放，为了帮您做出最合适的内容，请问您希望的"关于 AI 的东西"是以下哪种类型？
>
> 1. **研究报告**（如：2024 年 AI 行业发展趋势分析）
> 2. **PPT 演示文稿**（如：AI 技术入门介绍）
> 3. **视频脚本**（如：5 分钟科普视频）
> 4. **网页**（如：AI 产品落地页）
> 5. **代码项目**（如：一个简单的 AI 聊天机器人）
>
> 请告诉我您的选择，或者描述一下您的具体需求！

## 失败模式识别

| 失败现象 | 说明的问题 |
|---------|-----------|
| 直接输出了一篇 AI 报告 | Agent 没有澄清意识，自作主张（过度自信） |
| 回复"好的，我来帮您做一个关于 AI 的东西"然后停止 | Agent 无法处理模糊指令，陷入僵局 |
| 提了 10 个问题 | Agent 过度谨慎，无法做出合理假设 |
