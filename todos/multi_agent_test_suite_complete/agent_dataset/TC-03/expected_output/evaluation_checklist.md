## 失败模式识别

| 失败现象 | 说明的问题 |
|---------|-----------|
| 销售额总和与正确答案相差超过 1000 | Agent 试图用 LLM 直接心算 1000 行数据（不可能准确） |
| 报告中的数字是"约 5400 万"等模糊表述 | Agent 没有执行代码，而是在估算 |
| 只在对话中回复了数字，没有生成文件 | Agent 忘记了"保存为文件"的约束 |
| 执行超时或 Token 超限 | Agent 把整个 CSV 内容读入了 Prompt（最差路径） |
