# Deploy Directory

`deploy/` is the generated production deployment directory.

Current Docker development, release, deploy, data-directory, and troubleshooting
instructions live in:

- [OpenAgents Docker 开发与发版流程](../docs/guides/docker-compose-prod-selfhost-zh.md)

Tracked files in this directory are only documentation and templates. Runtime
files generated by `scripts/docker-deploy.sh` are ignored by git:

- `.env`
- `docker-compose.yml`
- `config.yaml`
- `gateway.yaml`
- `data/`
