## Filename Convention

| Action | Filename ID |
|--------|-------------|
| Happy waving | hi |
| Laughing hard | laugh |
| Crying tears | cry |
| Heart gesture | love |

## Custom Caption Guidelines

- Keep captions short: 1-3 words work best
- Actions auto-match caption meaning (e.g., "Sleepy" → yawning action)
- Users can provide captions in any language
