# Security Policy

## Supported Versions

As openagents doesn't provide an offical release yet, please use the latest version for the security updates.
Current we have two branches to maintain: 
* main branch for openagents 2.x
* main-1.x branch for openagents 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/openagents/security to report the vulnerability you find.
